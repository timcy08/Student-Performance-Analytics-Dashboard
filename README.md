🧠 Project Overview

This project focuses on analyzing student behavior and performance using synthetic data generated with Python. The goal is to transform raw activity data into meaningful insights using data preprocessing, exploratory data analysis (EDA), and an interactive Power BI dashboard.

The dashboard helps track student engagement, learning patterns, and performance trends across multiple dimensions such as login activity, course progress, events, and webinars.

🎯 Objectives
Generate realistic student activity data using Python (Faker library)
Combine multiple data sources into a unified student performance dataset
Perform data cleaning and exploratory data analysis (EDA)
Identify patterns in student engagement and academic performance
Build an interactive Power BI dashboard for visualization and insights
🛠️ Tech Stack
Python
Pandas
Faker (Synthetic Data Generation)
Jupyter Notebook
Power BI
CSV Data Processing


📊 Dashboard Features
Student performance tracking
Engagement analysis across platforms
Attendance and participation trends
Course progress visualization
Comparative student insights
KPI-based summary metrics
🔍 Key Insights
Identified high and low engagement students based on login activity
Measured correlation between course completion and participation
Analyzed webinar and event attendance impact on performance
Created segmentation of students based on activity levels
📈 Outcome

This project demonstrates how raw behavioral data can be converted into actionable insights using data analytics and visualization tools. It strengthens understanding of data pipelines, EDA, and dashboard development.
